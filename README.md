# 📌 Sobre o Projeto

O **ShoppingCart** é uma aplicação web que demonstra o funcionamento de um **carrinho de compras de e‑commerce**.

O projeto permite simular o fluxo básico de compra em uma loja online, incluindo **visualização de produtos, adição ao carrinho e gerenciamento de itens**.

> 🎯 O objetivo do projeto é praticar conceitos fundamentais de **JavaScript, manipulação do DOM e lógica de aplicações web interativas**.

---

# ✨ Funcionalidades

| Recurso | Descrição |
|---|---|
| 🛍️ **Catálogo de Produtos** | Exibição de produtos disponíveis |
| ➕ **Adicionar ao Carrinho** | Inclusão de itens no carrinho |
| 🔢 **Controle de Quantidade** | Atualização da quantidade de produtos |
| ❌ **Remover Itens** | Exclusão de produtos do carrinho |
| 💰 **Cálculo Automático** | Atualização dinâmica do total da compra |
| ⚡ **Atualização em Tempo Real** | Interface atualizada dinamicamente com JavaScript |

# 🚀 Como Executar

## Pré-requisitos

- Navegador moderno (Chrome, Edge, Firefox)
- Git instalado (opcional)

---

## Passo a Passo

```bash
# 1. Clone o repositório
git clone https://github.com/RDEsley/ShoppingCart.git

# 2. Acesse a pasta do projeto
cd ShoppingCart
```

Abra o arquivo:

index.html

em seu navegador.
